Marvic - The PHP Web Application Framework
==========================================

```php
$app = Marvic();

$app->get(function($request, $response) {
	$response->render('Welcome to Marvic!!!');
});

$app->run();
```

What's Marvic?
--------------

Marvic is a **minimalist PHP framework** with expresive and elegant syntax, that helps coders to build **minimalist and extensive web applications and API´s**, thinking on scalability of the project. Marvic is perfect to build **MVC based application** using the monolith architecture, but the coder is free to use other ar add dependences for your project.
